# WebOC Automation

This project automates data scraping and Excel exports from the WebOC system.

## Project Description

- `automation.py` → main automation script
- `util.py` → helper functions
- `automation_function.py` → additional automation logic (to be recreated if missing)
- `gui.py` → graphical interface (to be recreated if missing)
- `weboc.py` → main workflow script (to be recreated if missing)
- `exports/` → contains test/sample Excel files
- `output/` → stores generated files (ignored by Git)
- `.gitignore` → ignores virtual environments, caches, outputs, and other unnecessary files
- `requirements.txt` → lists Python dependencies
